# Student Report System using Pandas

## Project Overview

This project demonstrates basic data cleaning and analysis using the Pandas library in Python. It performs several common data preprocessing tasks on a student dataset and exports the cleaned results into CSV and JSON formats.

## Features

- Create a student dataset using Pandas
- Clean missing marks using the average value
- Add a Pass/Fail result column
- Filter students scoring 80 or above
- Calculate course-wise average marks
- Sort students by marks (highest to lowest)
- Export the final report to CSV
- Export the final report to JSON

## Technologies Used

- Python
- Pandas

- student_report.json
